# GPT-Chatbot

## Overview
This is a semi-standard chatbot that implements the [OpenAI API](https://platform.openai.com/docs/introduction), demonstrating how AI can be leveraged to create engaging and informative conversational interfaces. This showcases the basic capabilities of OpenAI's vast language model and its relevancy in various fields as a dynamic response service to users' questions.

## Features
- **Intelligent Responses**: Utilizes advanced NLP from OpenAI to understand and respond to user queries appropriately
- **Customizable**: Presents users with the option to adjust response style based on specific needs

## Stack
- React v18.2 + Vite v4.2
### Notable Dependencies
- chatscope/chat-ui-kit-react v1.10
- dotenv v16.4
- react-dom v18.2

## Setup
Open the Terminal where you want this project to exist and enter the following commands:
```
git clone https://github.com/AnvayB/GPT-Chatbot.git
cd gpt-chatbot
npm install
make start
```
